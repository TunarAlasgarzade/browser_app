# Browser App

A simple, privacy-focused browser built with Flutter. Uses DuckDuckGo as the default search engine.

## Features

- URL bar with smart detection (URL vs search query)
- DuckDuckGo search integration
- Back, forward, and reload navigation
- Home button to clear the page
- Android back gesture support (navigates back in history)
- Loading progress indicator
- Clean, minimal UI

## Tech Stack

- Flutter
- webview_flutter
